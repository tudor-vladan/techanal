## 🚀 **QUICK START**

### **Prerequisites**
- Docker și Docker Compose
- Node.js 18+ (pentru development)
- pnpm package manager

### **1. Clone Repository**
```bash
git clone https://github.com/yourusername/techanal.git
cd techanal
```

### **2. Environment Setup**
```bash
cp env.example .env
# Editează .env cu API keys pentru AI services
```

### **3. Start with Docker**
```bash
# Development
./scripts/docker-start.sh

# Production
./scripts/docker-prod.sh
```

### **4. Access Application**
- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:8787
- **Database**: PostgreSQL pe portul 5432

## 🛠️ **DEVELOPMENT**

### **Install Dependencies**
```bash
# Frontend
cd ui && pnpm install

# Backend
cd server && pnpm install

# Database
cd database-server && pnpm install
```

### **Development Scripts**
```bash
# Start development environment
pnpm run dev

# Run tests
pnpm run test

# Build for production
pnpm run build

# Docker management
./scripts/docker-start.sh    # Start development
./scripts/docker-prod.sh     # Start production
./scripts/docker-stop.sh     # Stop all services
```
